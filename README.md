# 📡 ProfRadar — PhD Position Scout

A personal AI-powered dashboard to discover recently hired professors at top US universities working in **VLMs**, **Computer Vision**, **AI**, and **Robotics**.

Built for **Rongali Sai Bhargav** · IIT Bombay · Honda Research Institute

---

## Features

- 🔍 **AI-powered search** — Uses Claude + web search to find new faculty in real-time
- 🎯 **Match scoring** — Ranks professors by fit with your research profile (VLMs, Robotics, Action Prediction)
- 🏫 **Multi-university filter** — MIT, Stanford, CMU, Berkeley, UW, UIUC, Michigan, Georgia Tech, and more
- ⭐ **Bookmark & track** — Save professors for follow-up (persisted in localStorage)
- 📬 **Quick outreach** — Direct email links to contact professors
- 📊 **Sort & filter** — By match score, join year, or name

## Usage

1. Open `index.html` in any modern browser (or visit the [GitHub Pages site](https://Saibhargav1208.github.io/ProfScraping))
2. Enter your [Anthropic API key](https://console.anthropic.com) (stored in session only, never leaves your browser)
3. Select target universities and research areas
4. Click **◈ SCAN PROFESSORS**

## Research Focus

The tool is tuned for PhD applicants working in:
- Vision-Language Models (VLMs)
- Self-supervised video learning
- Action prediction
- Computer Vision
- AI & Robotics

## Tech Stack

- Pure HTML/CSS/JS (no build step, zero dependencies)
- [Anthropic API](https://docs.anthropic.com) with web search for live faculty discovery
- Google Fonts: Bebas Neue + Cormorant Garamond + Fira Code
